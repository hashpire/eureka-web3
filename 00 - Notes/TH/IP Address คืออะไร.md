# IP Address คืออะไร
**IP Address** คือหมายเลขประจำเครื่องคอมพิวเตอร์แต่ละเครื่องในระบบเครือข่าย
- **วัตถุประสงค์** เพื่อใช้ระบุต้นทางและปลายทางในการติดต่อสื่อสารรับส่งข้อมูล
- **IP Address** มี 2 มาตรฐาน คือ
	- [[IPv4 คืออะไร|IPv4]]
	- [[IPv6 คืออะไร|IPv6]]
